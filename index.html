「Webコンテンツ」の成果

本テーマについて

情報工学の専門科目で学んだ基礎知識を活用して、それらを実際の問題解決に実践できる能力を修得できるようにする。また、HTMLなどのWeb技術を活用し、目的に応じたWebページを作成・設計できる力を身につけるようにする。

プロジェクト課題について

プロジェクト課題で作成したページ：

project-html

数当てゲーム

この数当てゲームは1~10の中から好きな数字を選択して、3回以内にコンピュータが決めた数字を当てるゲームです。数字を当てると正解のメッセージが表示され、間違えると間違いのメッセージとともにコンピュータが決めた数字より大きいか小さいかのメッセージが表示されます。また3回間違えると答えが表示され、4回以上回答すると、ゲームが終わっているというメッセージが表示されます。

kazuate.html

各授業回の学習要項

第1回 WWW と HTML

HTMLテキストの要素とハイパーリンクと画像の埋め込みなどを使ってWebコンテンツの作成をした。これから行うプロジェクト課題を実施するにあたって3種類のデータのうちどのデータのWebアプリケーションを作成するかを決める。

演習の成果： 演習1-4:HTMLのさまざまな要素

第2回 CSS による web ページの装飾

今回はWebページの色・配置・サイズ・レイアウトなどを指定するCSSという言語を学んだ。自分の作ったWebページを好みに表現できるため創造性を発揮できる点が大きな特徴である。自分もこれを使いこなせるようになりたい。

演習の成果： 演習2-3:CSSの使い方

第3回 JavaScript の基本

今回はWebページ上で複雑な機能を実現するJavaScriptという言語を学んだ。入力内容に応じて動作を切り替えるなどWebページにさまざまな動作をつけることができた。Webページがどのように動いているのかを体験的に理解することができた。

演習の成果： 演習3-6:オブジェクトの参照

第4回 DOM と JavaScript

今回はDOMとJavaScriptを組み合わせることで、ユーザーの操作に応じて変化する動きのあるWebページを作成した。ボタンのクリックや入力内容に応じて表示を変更する仕組みを実現した。この制作を通じて、DOM操作の基本的な仕組みとJavaScriptによるイベント処理の重要性を理解することができた。

演習の成果： 演習4-10:足し算ページ
