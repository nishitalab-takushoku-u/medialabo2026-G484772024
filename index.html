「Webコンテンツ」の成果

本テーマについて

情報工学の専門科目で学んだ基礎知識を活用して、それらを実際の問題解決に実践できる能力を修得できるようにする。また、HTMLなどのWeb技術を活用し、目的に応じたWebページを作成・設計できる力を身につけるようにする。

プロジェクト課題について

私が作ったプロジェクト課題は一部の国の名前を選択することでその国の緯度や経度、天気の情報を見られるサイトを作成した。工夫したことは検索のところをプロダウンメニューにしてそのメニューの中から国を選んで検索するようにした。また検索ボタンを押すと緯度や経度といった情報が出るようになり、それと同時に天気の画像が出るようになっていて、違う国を選択するたびに毎回変わるようにした。あとはcssの要素として色や点線をつけたりして華やかな感じに仕上げた。

プロジェクト課題で作成したページ： project-html

数当てゲーム

この数当てゲームは1~10の中から好きな数字を選択して、3回以内にコンピュータが決めた数字を当てるゲームです。まず初めに検索する枠の中に1~10の内好きな数字を入れて検索ボタンを押してください。正解すると正解のメッセージが表示されます。間違えると間違いのメッセージとともにコンピュータが決めた数字より大きいか小さいかのメッセージが表示されます。また3回以内に正解できなかったもしくは4回以上回答すると答えとともにゲーム終了のメッセージが表示されます。そのほかにコンピュータが決める数字はランダムになるように設定されています。また自分で検索する他に検索欄の右側にある矢印を使って数字を選ぶこともできます。

kazuate.html

各授業回の学習要項

第1回 WWW と HTML

HTMLテキストの要素とハイパーリンクと画像の埋め込みなどを使ってWebコンテンツの作成をした。これから行うプロジェクト課題を実施するにあたって3種類のデータのうちどのデータのWebアプリケーションを作成するかを決める。

演習の成果： 演習1-4:HTMLのさまざまな要素

第2回 CSS による web ページの装飾

今回はWebページの色・配置・サイズ・レイアウトなどを指定するCSSという言語を学んだ。自分の作ったWebページを好みに表現できるため創造性を発揮できる点が大きな特徴である。自分もこれを使いこなせるようになりたい。

演習の成果： 演習2-3:CSSの使い方

第3回 JavaScript の基本

今回はWebページ上で複雑な機能を実現するJavaScriptという言語を学んだ。入力内容に応じて動作を切り替えるなどWebページにさまざまな動作をつけることができた。Webページがどのように動いているのかを体験的に理解することができた。

演習の成果： 演習3-6:オブジェクトの参照

第4回 DOM と JavaScript

今回はDOMとJavaScriptを組み合わせることで、ユーザーの操作に応じて変化する動きのあるWebページを作成した。ボタンのクリックや入力内容に応じて表示を変更する仕組みを実現した。この制作を通じて、DOM操作の基本的な仕組みとJavaScriptによるイベント処理の重要性を理解することができた。

演習の成果： 演習4-10:足し算ページ

第5回 イベント処理

今回は前回やったDOMの続きで、画像やリンクを追加したり、要素の属性を追加したり、変更したりした。また追加した要素を削除したりもした。最後にボタンを押すことで今まで追加した要素をWeb上に表示させる作業も行った。

演習の成果： 演習5-7:DOMプログラミング

感想

私はHTML、css、javascriptを使ってWebページの作成を行いました。初めはどのようにしていいのかわからなかった場面もあって不安になったこともありました。しかしSAや先生に聞いてみて、言われた通りやってみたらうまくできてとても安心な気持ちになりました。この授業でもっとも一番作業するのが大変だったのはプロジェクト課題でした。cssやjavascriptで行う機能を色々試していきました。次に私はこの授業の演習課題として1~10の数字の中から好きな数字を選択して、数を当てる数当てゲームを作りました。この課題で苦戦したと思ったのは正解と間違い、ゲーム終了の宣言を正しく表示するところです。これはSAの人に聞いて色々と試した結果なんとか出来上がりました。私がもっとも悩ませたのはWebコンテンツ作成の活動記録とプロジェクト課題にcssの装飾を載せることです。色や文字の形を自分の好きなようにデザインしてもいいと言われてもどのようにしたら良いか迷いました。悩んでいても仕方なかったので色々と試して自分なりにアレンジしてみました。結果自分としてはかなり満足できる仕上がりになりました。次に私が工夫した点はproject課題の画像を表示させるところです。天気情報で調べたい国を選ぶと緯度や経度とかの情報が表示されるほかに天気の画像を表示させるようにしました。また検索キーワードのところプロトダウンメニューにして検索しやすいようにしました。私はそれとそれぞれの国の都市名の地図をいれようと思いましたが、できなかったので断念しました。次に私が強く注いだところはWebコンテンツ作成の活動記録の題名の文字の色をそれぞれ別の色にして分けて、その内容を四角の枠で囲んで読みやすくしたことです。そうすることで整理されたように感じました。最後に私がWebコンテンツ作成の授業で学んだことを話したいと思います。私が幼い頃から見たことあるWebページがどのように作られているか考えたこともなかったです。この授業でHTML、css、javascriptの機能を使いどのようにして動いているのかを知っただけでなく、自分でWebページを作るといった体験ができました。またWebページの内容でどのような色や形にしていくかといったアレンジ力を身につけられました。もし私が将来、Webページを作成するようになったらこの授業で習ったことを活かして取り組んでいきたいと思っています。
